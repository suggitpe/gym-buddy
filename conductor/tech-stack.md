# Tech Stack

**Architecture**: Kotlin-Multiplatform (KMP)

## Key Technologies (MVP Phase 1):
- **Shared Logic**: Kotlin commonMain
- **Database**: SQLDelight

## Future Technologies (Beyond MVP Phase 1):
- **AI Engine**: Gemini 3 Flash (Remote) + Whisper (Local fallback) (For voice activation)
- **Biometrics**: Fitbit Web API + Health Connect (Android) (For external data source integration)
- **Audio Engine**: Native AVAudioSession (iOS) / AudioManager (Android) (For audio features)
